% operator Link to heading

  • %s: String conversion.
  • %d or %i: Integer conversion.
  • %f: Float conversion.
  • %o: Octal conversion.
  • %x or %X: Hexadecimal conversion.
  • %e or %E: Exponential notation conversion.
In [1]: "%s %d %f %o %x %e" % ("a", 1, 1.0, 8, 16, 100)
Out[1]: 'a 1 1.000000 10 10 1.000000e+02'

  • padding
    val = "test"
f"{val:>10}"
# '      test'
f"{val:<10}"
# 'test      '
f"{val:_<10}"
# 'test______'
f"{val:^10}"
# '   test   '
