## Learned
- List active transactions in [PostgreSQL]({{< ref "PostgreSQL" >}})
   ```sql
   SELECT
       datname AS database,
       usename AS user,
       pid,
       application_name,
       client_addr,
       state,
       state_change,
       query
   FROM
       pg_stat_activity
   WHERE
       state = 'active' OR state = 'idle in transaction';
   ```
