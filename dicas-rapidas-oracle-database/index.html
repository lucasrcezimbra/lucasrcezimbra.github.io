Dicas rápidas Oracle Database

março 26, 2019
Dicas Rápidas
database • dicas • oracle

Criar um usuário para o banco de dados

CREATE USER my_user IDENTIFIED BY my_password;

Alterar a senha de um usuário já existente

ALTER USER my_user IDENTIFIED BY my_password;

Adicionar grants para um usuário do banco

GRANT 
    insert,
    select,
    update,
    delete
ON
    my_table
TO
    my_user;
